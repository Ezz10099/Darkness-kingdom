# Darkness Kingdom

Darkness Kingdom is a mobile portrait idle RPG / hero collector inspired by the AFK Arena core loop.

The project direction is dark fantasy, gothic kingdom wars, ancient magic, hero collection, idle progression, and premium mobile RPG presentation.

## Core identity

- Official title: Darkness Kingdom
- Genre: idle RPG / hero collector
- Combat: mostly auto battle with player-triggered ultimates
- Team structure: 5 heroes, front/back formation
- Setting: multiple ancient kingdoms, corrupted realms, royal wars, curses, abyssal magic, and resurrected rulers
- Player role: the resurrected ruler of a fallen dark kingdom

## Naming authority — important

This project is **Darkness Kingdom**.

Do not use, restore, or introduce the old **Arcane Academy** identity anywhere in code, UI, docs, save names, asset names, prompts, lore, onboarding text, or generated assets.

Also avoid academy/school/classroom terminology unless the user explicitly asks for it. The old academy direction has been fully replaced by the dark kingdom direction.
