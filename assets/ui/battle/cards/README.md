# Battle card assets

Place transparent PNG card UI pieces here.

Recommended files:

- `battle_card_frame_idle.png`
- `battle_card_frame_selected.png`
- `battle_card_frame_ready_glow.png`
- `battle_card_charge_bg.png`
- `battle_card_charge_fill.png`

Rules:

- Do not bake hero names into card images.
- Do not bake ultimate percentages into card images.
- Render hero names, level, and charge text in Phaser code.
