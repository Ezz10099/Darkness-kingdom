# Battle button assets

Place transparent PNG battle button shells here.

Recommended files:

- `btn_battle_primary_idle.png`
- `btn_battle_primary_pressed.png`
- `btn_battle_primary_disabled.png`
- `btn_circle_idle.png`
- `btn_circle_active.png`
- `btn_circle_pressed.png`

Rules:

- Do not bake `Begin Battle`, `Pause`, `x2`, or `Auto` text into the image.
- Generate empty button/frame art.
- Render labels and state text in Phaser code.
