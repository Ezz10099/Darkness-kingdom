# Hero Assets

This folder organizes uploaded hero PNG assets.

- `portraits` = hero card images
- `full` = full-screen hero detail images
- `battle` = battle scene images

## Filename format

Use the same `hero_id` across all three versions:

- `hero_id_portrait.png`
- `hero_id_full.png`
- `hero_id_battle.png`

## Requirements

- Keep the same `hero_id` across portrait/full/battle images for each hero.
- Use compressed PNG files with transparency.
