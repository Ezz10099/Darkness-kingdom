# Project Identity Note

## Official project name

The official title is **Darkness Kingdom**.

## Non-negotiable naming rule

Do not use the old name **Arcane Academy** anywhere in this project.

This applies to:

- visible UI text
- browser title
- preload/loading text
- onboarding/tutorial text
- stage names
- hero lore
- save keys, except legacy migration fallback only
- asset names
- folder names
- docs
- prompts for AI-generated assets
- future PR titles and implementation notes

## Theme rule

Avoid academy, school, classroom, student, headmaster, training yard, lesson, or similar educational identity unless the user explicitly asks for it.

The game identity is now:

- dark kingdom
- fallen crown
- resurrected ruler
- gothic fantasy
- royal purple magic
- antique gold UI
- corrupted realms
- idle RPG / hero collector

## Legacy exception

The only acceptable use of the old save name is a hidden technical fallback for migrating old localStorage saves. Do not show it to players and do not use it for new saves.
