# Darkness Kingdom — Design Constitution

## Game Identity
Darkness Kingdom is a mobile portrait idle dark fantasy RPG.

The world is a kingdom swallowed by darkness, ruled by ancient curses, shadow factions, corrupted castles, forbidden magic, and powerful heroes fighting through campaign stages, towers, guild wars, and summoned destinies.

## Core Loop
Collect idle rewards → upgrade heroes/gear → push campaign → hit wall → use side modes → earn resources → summon/ascend → break wall → unlock systems → repeat.

## Inspiration Boundary
The game may use AFK Arena-style idle RPG structure:
- idle rewards
- hero collection
- auto battles
- campaign progression walls
- summoning
- ascension
- towers
- arena
- guild
- daily/weekly quests
- event rewards

The game must NOT copy:
- AFK Arena characters
- AFK Arena names
- AFK Arena exact UI screens
- AFK Arena art
- AFK Arena icons
- AFK Arena text
- AFK Arena faction identities

## Visual Identity
Dark Arcane Premium.

Black depth, royal purple magic, antique gold trim, crimson danger/event accents, gothic panels, ornate borders, crystal resources, premium fantasy gacha presentation.

## UI Rule
All screens must use a reusable Darkness Kingdom UI kit before individual screen redesign.

Reusable components:
- premium panel
- ornate button
- top resource bar
- bottom navigation
- icon button
- hero card
- reward card
- tab button
- notification badge
- modal overlay

## Naming Rule
Avoid academy/school/student/classroom/magic lesson terminology unless explicitly approved.

Use kingdom/darkness/shadow/cursed/royal/guild/tower/realm/ancient/void/abyss terminology.
